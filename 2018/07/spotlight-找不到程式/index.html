Spotlight 找不到程式

因為可以綁快速鍵，一直以來我都是以 Spotlight 開啟 iTerm，但是今天更新之後，在 Spotlight 搜尋卻怎麼樣都找不到 iTerm，簡直就是災難！（有沒有這麼誇張）

找了一下解決辦法發現超簡單，關掉再重開就好了，記錄一下作法：

# 關掉 Spotlight:
sudo mdutil -a -i off

# 重新開啟 Spotlight:
sudo mdutil -a -i on
