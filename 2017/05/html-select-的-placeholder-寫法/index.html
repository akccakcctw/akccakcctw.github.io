HTML select 的 placeholder 寫法

該如何讓select表單預設顯示在placeholder選項，但此選項僅作為提示用，不能選取呢？

可以這樣寫：

<select>
    <option value="" disabled selected>請選擇</option>
    <option value="a">a</option>
    <option value="b">b</option>
</select>

如此一來，就會預設選在「請選擇」的位置，且 user 不能選這個選項
