各種符號的英文讀法

運算符號

Symbol Description 中文說法
+ Plus, And 加號、正號
- Minus, Negative 減號、負號
* Times, Multiplied by 乘號
/ Devided by 除號
% Percent 百分號
= Equals 等號
( ) Open(close) parenthesis, Parentheses 小括號
[ ] Open(close) bracket, Square brackets 方括號
{ } Open(close) brace, Open(close) curly, Curly brackets 大括號

Symbol Description 中文說法
> Greater than 大於
< Less than 小於

== Equal to 等於
!= Not equal to 不等於
=== Equivalent to 全等於
&& AND 且
|| OR 或
